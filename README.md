# 📄 AI Document Analyzer

An intelligent document analysis tool built with Python and Streamlit. This application allows users to upload complex research papers and instantly extracts high-level summaries and key insights using advanced LLM processing.

## 🚀 Features
* **PDF Text Parsing:** Seamlessly extracts raw text from dense academic and technical PDFs.
* **Instant AI Analysis:** Processes text to generate concise, professional summaries (e.g., explaining complex mechanisms like Proof-of-Authority).
* **Interactive UI:** Built with Streamlit for a clean, responsive, and user-friendly web interface.
* **Demo Mode:** Includes a local bypass toggle for demonstration purposes without triggering live API quotas.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Frontend:** Streamlit
* **Document Processing:** PyPDF2 (or your specific PDF library)
* **AI Integration:** Google Gemini API (Configurable)

## ⚙️ How to Run Locally
1. Clone the repository: `git clone [your-repo-link]`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `streamlit run app.py`
